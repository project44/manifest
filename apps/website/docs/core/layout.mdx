Spacing tokens are used for padding, margins, and position coordinates.

The Manifest Design System spacing scale is based off 4px increments, anything within that range is
supported. Below is a collection of our most commonly used spacing tokens.

| Name           | Size |
| -------------- | ---- | ---------------------------------------------------------------------------------------------------- |
| $space-x-small | 4px  | <div style={{ backgroundColor: 'var(--color-palette-indigo-50)', height: '4px', width: '4px' }} />   |
| $space-small   | 8px  | <div style={{ backgroundColor: 'var(--color-palette-indigo-50)', height: '8px', width: '8px' }} />   |
| $space-medium  | 16px | <div style={{ backgroundColor: 'var(--color-palette-indigo-50)', height: '16px', width: '16px' }} /> |
| $space-large   | 24px | <div style={{ backgroundColor: 'var(--color-palette-indigo-50)', height: '24px', width: '24px' }} /> |
| $space-x-large | 32px | <div style={{ backgroundColor: 'var(--color-palette-indigo-50)', height: '32px', width: '32px' }} /> |

Label, categorize, or organize items using keywords that describe them.

```jsx live
<Tag>Tag</Tag>
```

## Usage

### Removable

Set the `isRemovable` prop to render a remove button within the tag.

```jsx live
<Tag isRemovable>Tag</Tag>
```

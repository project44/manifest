---
title: Tooltip
description: Display additional context when users hover or focus an element.
---

```jsx
<Tooltip title="This is a tooltip">
  <Button>Hover Me</Button>
</Tooltip>
```

## Examples

### Placement

Set the `placement` prop on tooltip to adjust the placement of tooltip in relation to the trigger
element.

```jsx
<Tooltip placement="right" title="This is a tooltip">
  <Button>Hover Me</Button>
</Tooltip>
```

### Delay

The delay in which the tooltip displays can be adjusted with the `delay` prop.

```jsx
<Tooltip delay={500} title="This is a tooltip">
  <Button>Hover Me</Button>
</Tooltip>
```

### Controlled

The pagination's state can be controlled using the `isOpen` and `onOpenChange` props.

```jsx
function ControlledExample() {
  const [open, setOpen] = React.useState(false);

  return (
    <Tooltip isOpen={open} onOpenChange={setOpen} title="Tooltip">
      <Button>Controlled</Button>
    </Tooltip>
  );
}
```

## Accessibility

Tooltip triggers must be focusable and hoverable in order to ensure that all users can activate
them.

---
title: Tag
description: Label, categorize, or organize items using keywords that describe them.
---

```jsx
<Tag>Tag</Tag>
```

## Examples

### Removable

Set the `isRemovable` prop to render a remove button within the tag.

```jsx
<Tag isRemovable>Tag</Tag>
```

import { Tag } from '@project44-manifest/react';

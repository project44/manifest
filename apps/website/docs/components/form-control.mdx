---
title: Form Control
description: Attach a label and helper text to form inputs.
---

```jsx
<FormControl label="Email" helperText="Please enter your email address">
  <input />
</FormControl>
```

## Examples

### Optional

By default form control elements are required, to mark form element as optional set the `isRequired`
prop to false.

```jsx
<FormControl label="Email" helperText="Email is required" isRequired={false}>
  <input />
</FormControl>
```

### Invalid State

Set the `validationState` prop to `invalid` to render the form control in an invalid state.

```jsx
<FormControl label="Email" helperText="Email is required" validationState="invalid">
  <input />
</FormControl>
```

## Accessibility

### Labeling

If a visible label isn't specified, an aria-label must be provided to the input for accessibility.
If the field is labeled by a separate element, an aria-labelledby prop must be provided using the id
of the labeling element instead.

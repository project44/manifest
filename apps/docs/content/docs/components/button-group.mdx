---
title: Button Group
description: Group related buttons.
---

```jsx
<ButtonGroup variant="secondary">
  <Button>Button</Button>
  <Button>Button</Button>
</ButtonGroup>
```

## Examples

### Attached

Buttons within a ButtonGroup can be attached to each other.

```jsx
<ButtonGroup isAttached>
  <Button>Button</Button>
  <Button>Button</Button>
</ButtonGroup>
```

## API

### Import

```js
import { ButtonGroup } from '@project44-manifest/react';
```

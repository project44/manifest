---
title: Separator
description: Divide content in close proximity.

```jsx
<Separator />
```

## Examples

### Vertical

Set the `orientation` prop to adjust the layout of the separator.

```jsx
<Flex css={{ size: 50 }}>
  <Separator orientation="vertical" />
</Flex>
```

### Import

```js
import { Separator } from '@project44-manifest/react';
```

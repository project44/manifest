title: Box
description: A polymorphic component used for access theme aware styling.
---

## Examples

### CSS Prop

All Manifest Design System components include a css prop, which allows for theme aware styles.

```jsx
<Box css={{ backgroundColor: '$palette-indigo-500', size: 300 }} />
```

### As prop

All components a polymorphic `as` prop. This prop is useful for changing the rendering element of a component. You can also use the as prop to render another component.

```jsx
<Box
  as="button"
  css={{
    all: 'unset',
    backgroundColor: '$palette-indigo-500',
    color: '$palette-white',
    borderRadius: '$full',
    px: '$medium',
    py: '$small',
  }}
>
  Button
</Box>
```

## API

### Import

```js
import { Box } from '@project44-manifest/react';
```

### Props

Props of the native element are also available.

| Prop      | Type              | Description                   | Default  |
| --------- | ----------------- | ----------------------------- | -------- |
| as?       | `string`          | The HTML tag to be rendered.  | `button` |
| children? | `React.ReactNode` | The content of the component. |          |
| css?      | `CSS`             | Theme aware style object.     |          |

---
title: Link
description: Navigate a user to another page or website.
---

```jsx
<Link href="https://manifestdesignsystem.com">Manifest Design System</Link>
```

## Examples

### Inline

Links can be rendered inline with text.

```jsx
<Typography>
  Visit the manifest design system <Link href="https://manifestdesignsystem.com">website</Link>
</Typography>
```

## API

### Import

```js
import { Link } from '@project44-manifest/react';
```

All native element props are also available.

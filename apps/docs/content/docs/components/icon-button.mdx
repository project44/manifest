---
title: IconButton
description: Clickable elements that communicate actions that users can take.
---

```jsx
<IconButton>
  <Icon icon="add" />
</IconButton>
```

## Examples

### Primary

Primary buttons are used to indicate the highest priority action a user can perform.

```jsx
<IconButton variant="primary">
  <Icon icon="add" />
</IconButton>
```

### Secondary

Secondary buttons are second in priority to primary buttons and are used for important actions that
aren't the primary action.

```jsx
<IconButton variant="secondary">
  <Icon icon="add" />
</IconButton>
```

### Tertiary

Tertiary buttons are the lowest priority actions and should be used for less commonly used actions.

```jsx
<IconButton variant="tertiary">
  <Icon icon="add" />
</IconButton>
```

### Destructive

Destructive buttons are used to indicate an action that is destructive and cannot be undone.

```jsx
<IconButton variant="danger">
  <Icon icon="add" />
</IconButton>
```

### Brand

Brand buttons are used for project44 specific marketing moments and should be use sparingly.

```jsx
<IconButton variant="brand">
  <Icon icon="add" />
</IconButton>
```

### Small

Small icon buttons are used when vertical spacing is limited.

```jsx
<IconButton size="small">
  <Icon icon="add" />
</IconButton>
```

### Disabled State

Set the `isDisabled` prop to prevent a user from pressing a button.

```jsx
<Stack gap="medium" orientation="horizontal">
  <IconButton isDisabled>
    <Icon icon="add" />
  </IconButton>
  <IconButton isDisabled variant="secondary">
    <Icon icon="add" />
  </IconButton>
  <IconButton isDisabled variant="tertiary">
    <Icon icon="add" />
  </IconButton>
  <IconButton isDisabled variant="danger">
    <Icon icon="add" />
  </IconButton>
  <IconButton isDisabled variant="brand">
    <Icon icon="add" />
  </IconButton>
</Stack>
```

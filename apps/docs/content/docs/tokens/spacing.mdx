---
title: Spacing
description: Spacing tokens are used for padding, margins, and position coordinates.
---

The Manifest Design System spacing scale is based off 4px increments, anything within that range is supported. Below is a collection of our most commonly used spacing tokens.

| Name    | Space | Token       |                                                                               |
| ------- | ----- | ----------- | ----------------------------------------------------------------------------- |
| x-small | 4px   | spaceXSmall | <Box css={{ backgroundColor: '$palette-indigo-50', height: 8, width: 4 }} />  |
| small   | 8px   | spaceSmall  | <Box css={{ backgroundColor: '$palette-indigo-50', height: 8, width: 8 }} />  |
| medium  | 16px  | spaceMedium | <Box css={{ backgroundColor: '$palette-indigo-50', height: 8, width: 16 }} /> |
| large   | 24px  | spaceLarge  | <Box css={{ backgroundColor: '$palette-indigo-50', height: 8, width: 24 }} /> |
| x-large | 32px  | spaceXLarge | <Box css={{ backgroundColor: '$palette-indigo-50', height: 8, width: 32 }} /> |
